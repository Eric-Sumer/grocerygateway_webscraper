How to use this Repo:

1) Pull Repo into Computer
2) Python 3 must be installed in the computer (go to python website to do so)
3) install the following with the command "pip3 install <library_in_list>"
- selenium
4) run the file with command "python3 generalWebscraper.py" in the folder in which the file is in 
- this needs to be done in terminal (read some guides of how to navigate within the computer with terminal//command line)

All the products will be generated in the productList.csv file. On each new run, productList is deleted and rewritten.

DO NOT EDIT PRODUCTLIST DIRECTLY

To make edits, make a copy of productList and make edits in the copy of productList that was made. Edits include applyuing filters, changing column arrangements, etc
